Hugo + GitHub Pages：我的第一个静态博客诞生记

本文记录了如何从零开始，使用 Hugo 静态网站生成器和 GitHub Pages 免费服务，搭建一个属于自己的、快速且安全的个人博客。

为什么选择静态博客？

在当今这个时代，我们有无数种方式来创建博客，比如 WordPress、Ghost 等动态 CMS。但我最终选择了 Hugo，一个用 Go 语言编写的静态网站生成器。主要原因有三：

  1. 速度：快，无与伦比的快！生成的纯 HTML 页面无需数据库查询，加载速度极快。
  2. 安全：没有数据库，没有动态后端，几乎杜绝了常见的网站攻击漏洞。
  3. 成本：我可以将它托管在 GitHub Pages 或 Netlify 上，完全免费。

核心步骤回顾

搭建过程可以总结为以下几个关键步骤。

1. 环境安装

首先，你需要安装 Git 和 Hugo。在 macOS 上，一条命令就能搞定：

brew install hugo

2. 创建站点和选择主题

创建新站点非常简单：

hugo new site my-blog
cd my-blog
git init
# 添加一个喜欢的主题
git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

3. 撰写内容

这正是我现在正在做的事情！使用 Markdown 语法，我可以专注于内容创作本身，而不用分心于排版。例如，我可以轻松地创建一个列表：

  • 优点一：专注写作
  • 优点二：版本控制
  • 优点三：跨平台兼容

"代码和文字一样，都是思想的表达。"

总结

从决定到完成，整个过程不到一个小时。拥有一个完全由自己掌控的博客感觉真的很棒。接下来，我计划在这里持续分享我的学习笔记和项目经验。
